# Templates Directory

This directory contains medical report templates organized by anatomical region.

## Structure

```
templates/
├── chest/      # Chest X-ray and thoracic imaging templates
├── brain/      # Brain MRI/CT imaging templates
├── abdomen/    # Abdominal imaging templates
└── spine/      # Spine imaging templates
```

## Usage

Each subfolder contains JSON template files that define structured report formats for medical imaging analysis.

## Template Format

Templates are JSON files that define:
- Report structure
- Required fields
- Formatting guidelines
- Standard terminology
